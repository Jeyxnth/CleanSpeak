🎙️ CleanSpeak

CleanSpeak is an AI-based audio processing pipeline that cleans noisy audio, converts speech to text, and filters inappropriate content.

🚀 Features
    1.Noise reduction (Librosa, Noisereduce)
    2.Speech-to-text using Whisper
    3.Profanity filtering
    4.Modular pipeline design

⚙️ Workflow

Audio → Noise Cleaning → Transcription → Word Filtering → Clean Output

setup - Run:

  git clone https://github.com/Jeyxnth/CleanSpeak.git
  cd CleanSpeak
  pip install -r requirements.txt
